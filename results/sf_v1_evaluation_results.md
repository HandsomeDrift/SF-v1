| **Vid 50-way** | 0.3177 | 0.0073 | ✗ 随机水平 |
| **Hue-PCC** | 0.4103 | 0.3817 | ↓ |
| **VIFI-Score** | 0.8485 | 0.5871 | ✗ |
| **CLIP-PCC** | 0.9753 | 0.2167 | ✗ 崩溃 |

### 评估统计
- **样本数**: 483/540 (57 missing at eval time)

### 分析
v1 呈现极端割裂的模式：
- **运动/时序指标**（EPE, CTC, DTC）反而优于 baseline — Fast Branch 的时序动态捕获有效
- **语义/像素/视频质量指标**全面崩溃 — DiT 无法适配 SF guidance

**根因**: LoRA scaling=1/128 导致 DiT 几乎不更新权重，无法学会如何利用 SF guidance 信号。Guidance 注入反而干扰了去噪过程，生成的视频时序流畅但内容完全偏离 GT。

**结论**: LoRA scaling 必须足够大让 DiT 真正适配 guidance，v2 的 scaling=1.0 是正确选择。

### 5.4 跨被试分析

1. **FVD 全面大幅改善**: m05→d03 ↓16%, m05→d04 ↓35%, 三场景平均 ↓27%。证明 SF-v1 的视频质量优势在跨被试场景下同样成立。

2. **被试间差异显著**:
   - Sub-04 表现最好（FVD ↓35%, Img 50-way ↑15%, PSNR ↑11%）
   - Sub-03 的语义指标（Img/Vid 50-way）退步较大，但运动/结构指标（EPE ↓25%, SSIM ↑9%, DTC ↑1.5%）改善明显
   - 可能原因：Sub-03 和 Sub-05 的个体脑信号差异更大，影响语义解码但不影响时序动态捕获

3. **Vid 50-way 是主要弱项**: 三场景平均 ↓7.8%，说明视频级语义分类精度有待改善。可能与 LoRA 适配深度或 guidance 注入方式有关。

4. **Hue-PCC 持续退步**: 三场景平均 ↓5.6%，色彩重建是 SF-v1 的稳定弱点，需后续优化。

## 6. 核心结论

1. **Slow-Fast 假设验证成功**: EEG Fast Branch 显式编码时序动态，结合 fMRI Slow Branch 的语义信息，在视频重建质量上全面优于 unified fusion baseline。

2. **FVD 改善是论文级结果**: 同被试 ↓31%，跨被试平均 ↓27%，在脑信号视频重建领域是非常显著的提升。

3. **运动重建是 SF-v1 的核心优势**: 同被试 EPE ↓20%，跨被试 m05→d03 EPE ↓25%，直接验证了 Fast Branch 对运动信息的贡献。

4. **跨被试泛化能力强**: 模型仅在 Sub-05 上训练，泛化到 Sub-03/Sub-04 时 FVD 仍大幅优于各自的 baseline，说明 SF 架构学到的不是被试特异性特征。

5. **LoRA scaling 是关键超参**: scaling=1/128 导致灾难性失败，scaling=1.0 是正确选择。

6. **待完善项**:
   - Vid 50-way 跨被试退步 ↓7.8% 需要改善（可能通过改进 guidance 注入方式）
   - Hue-PCC 持续退步 ↓5.6% 需要分析（可能与色彩空间处理有关）
   - 推理优化消融实验进行中（alpha-Guidance + DANA）
